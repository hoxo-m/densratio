# Package index

## All functions

- [`KLIEP()`](http://hoxo-m.github.io/densratio/reference/KLIEP.md) :
  Estimate Density Ratio p(x)/q(x) by KLIEP (Kullback-Leibler Importance
  Estimation Procedure)
- [`RuLSIF()`](http://hoxo-m.github.io/densratio/reference/RuLSIF.md) :
  Estimate alpha-Relative Density Ratio p(x)/(alpha p(x) + (1-alpha)
  q(x)) by RuLSIF (Relative unconstrained Least-Square Importance
  Fitting)
- [`densratio()`](http://hoxo-m.github.io/densratio/reference/densratio.md)
  : Estimate Density Ratio p(x)/q(x)
- [`squared_euclid_distance()`](http://hoxo-m.github.io/densratio/reference/squared_euclid_distance.md)
  : Compute Squared Euclid Distance
- [`uLSIF()`](http://hoxo-m.github.io/densratio/reference/uLSIF.md) :
  Estimate Density Ratio p(x)/q(x) by uLSIF (unconstrained Least-Square
  Importance Fitting)
